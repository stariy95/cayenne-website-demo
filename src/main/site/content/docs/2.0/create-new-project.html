Create New Project

Start New Project. Select "File > New Project" menu item (or click "New Project" toolbar button):

A new project is created with a single empty DataDomain:

When you click on "UntitledDomain", this opens an editor for this domain and also activates two new actions: "Create DataNode" and "Create DataMap".
