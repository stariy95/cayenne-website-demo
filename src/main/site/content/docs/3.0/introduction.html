This book describes how to use Cayenne Persistence API to write persistent Java applications. It includes the following topics:

  • "Getting Started" tutorials,
  • Cayenne API concepts, such as ObjectContext, Persistent objects, Queries
  • Deployment
  • Tuning and performance optimization

See Also:

  • What's New in the Latest Release
  • Modeler Guide: A guide to object-relational mapping (ORM) with CayenneModeler GUI tool.
  • Remote Object Persistence Guide: Writing distributed Cayenne client applications.
